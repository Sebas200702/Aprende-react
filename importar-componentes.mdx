## Export: Compartiendo tu componente

Hay dos formas de exportar: **named export** y **default export**.

### Default Export (Exportaci√≥n por defecto)

**Un archivo, una exportaci√≥n principal:**

```tsx

  texto: string;

  return <button onClick={onClick}>{texto}</button>;

```

<Tip>
  Usa **default export** cuando el archivo tiene **un componente principal**. Es lo m√°s com√∫n para componentes React.
</Tip>

### Named Export (Exportaci√≥n nombrada)

**Un archivo, varias exportaciones:**

```tsx
  return `$${precio.toFixed(2)}`;
```

## Import: Usando componentes de otros archivos

## Rutas de importaci√≥n

### Rutas absolutas (con alias)

Puedes configurar alias en tu `tsconfig.json`:

Entonces puedes importar as√≠:


## Lazy loading (carga diferida)

<Info>
